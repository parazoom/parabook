# ParaBook 🪂

**Carnet de sauts (logbook) pour parachutistes** — simple, gratuit, sans publicité et qui fonctionne **hors-ligne**.

👉 **Application en ligne : https://parazoom.github.io/parabook/**

ParaBook permet d'enregistrer tous tes sauts et de suivre ta progression : zones de saut, matériel, types de saut, aéronefs, statistiques détaillées… Toutes tes données restent **sur ton appareil** (aucun compte, aucun envoi sur internet).

---

## ✨ Ce que fait l'application

- **Enregistrement des sauts** : numéro, date, zone de saut, type, aéronef, matériel (sac, voile, combinaison), libération, notes.
- **Résumé** : nombre total de sauts, libérations, zones et pays visités, dernier saut, activité sur 30 jours / 12 mois.
- **Totaux & statistiques** : répartition des sauts par zone, aéronef, matériel, type de saut et année.
- **Listes** : gère tes zones de saut, équipements, types de saut, aéronefs et liens utiles (avec archivage).
- **Import / Export CSV** : récupère ton ancien carnet ou sauvegarde le tien.
- **Personnalisation** : couleurs du thème, langue (🇫🇷 français / 🇬🇧 anglais).
- **100 % hors-ligne** une fois installée.

---

## 📲 Installation

Aucun store, aucun téléchargement compliqué : c'est une **application web installable (PWA)**.

### iPhone / iPad
1. Ouvre **https://parazoom.github.io/parabook/** avec **Safari**.
2. Touche le bouton **Partager** (le carré avec une flèche ↑).
3. Choisis **« Sur l'écran d'accueil »**, puis **Ajouter**.

### Android
1. Ouvre le lien avec **Chrome**.
2. Touche la bannière **« Installer l'application »**, ou menu **⋮** → **« Installer l'application »**.

### Ordinateur (Chrome / Edge)
Ouvre le lien, puis clique sur l'icône d'installation **⊕** dans la barre d'adresse.

L'icône ParaBook apparaît alors comme une vraie application et fonctionne même sans connexion.

---

## 🚀 Utilisation rapide

L'app s'organise en 5 onglets :

| Onglet | À quoi ça sert |
|---|---|
| **Résumé** | Vue d'ensemble de ton activité et tes statistiques clés. |
| **Sauts** | Liste de tous tes sauts. Le bouton **+** en ajoute un ; le bouton copier duplique le dernier. |
| **Totaux** | Statistiques détaillées par zone, aéronef, matériel, type et année. |
| **Listes** | Gestion de tes zones de saut, équipements, types, aéronefs et liens utiles. |
| **Actions** | Import/Export CSV, réglages (thème, langue), infos parachutiste, aide. |

> 💡 **Sauvegarde** : tes données sont stockées dans le navigateur de ton appareil (non synchronisées entre appareils). Pense à **exporter régulièrement** ton carnet via *Actions → Exporter en fichier* pour ne rien perdre.
